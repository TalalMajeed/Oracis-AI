## ğŸ—ï¸ **Setup and Installation**

### **Frontend Setup**

1. Clone the repository:

```bash
git clone https://github.com/OracisAI/oracis-frontend.git
```

2. Install dependencies:

```bash
cd oracis-frontend
npm install
```

3. Start the development server:

```bash
npm run dev
```

### **Backend Setup**

1. Clone the repository:

```bash
git clone https://github.com/OracisAI/oracis-backend.git
```

2. Install dependencies:

```bash
cd oracis-backend
npm install
```

3. Start the server:

```bash
npm start
```

### **AI Server Setup**

1. Clone the repository:

```bash
git clone https://github.com/OracisAI/oracis-ai.git
```

2. Install dependencies:

```bash
cd oracis-ai
pip install -r requirements.txt
```

3. Start the server:

```bash
uvicorn main:app --reload
```

### **Blockchain Setup**

1. Compile smart contracts using Truffle:

```bash
truffle compile
```

2. Deploy to local Ganache:

```bash
truffle migrate
```
